  • Map: returns an array of pieces of information from the original array. In the callback function, return the data you wish to be part of the new array.
  • Filter: returns a subset of the original array based on custom criteria. forEach: forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.
  • find: find() method returns the value of the first element that passes a test.

what is the difference between cookies,sessionStorage and localStorage ?

  • localStorage - stores data with no expiration date.
  • sessionStorage - stores data for one session and data will lost when the browser tab is closed.
  • Cookies - Cookies are text files with small pieces of data — like a username and password — that are used to identify your computer as you use a computer network. Specific cookies known as HTTP cookies are used to identify specific users and improve your web browsing experience.
what is event loop???

The event loop is a mechanism in JavaScript that manages the execution of tasks in an asynchronous, non-blocking manner. It continuously checks the call stack and the task queue, and when the call stack is empty, it moves tasks from the queue to the stack, where they are executed. This allows for efficient handling of I/O operations and prevents the application from becoming unresponsive. what is the work of quiz timer function?

The quizTimer function is responsible for setting a countdown timer for the quiz, displaying the remaining time to the user, and ending the quiz when the time is up. It uses the setInterval method to update the timer every second and calls the endQuiz function when the time runs out.

what is work of chooseQuiz function ?

chooseQuiz function allows the user to select a quiz to take by displaying a menu of available quiz options and prompting the user to input their choice. Once the user has made a selection, the function returns the corresponding quiz object, which contains information about the questions and answers for that quiz. If the user enters an invalid choice, the function prompts them to try again until a valid choice is made.
